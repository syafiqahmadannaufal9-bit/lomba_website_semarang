# Semarang Smart City Website

Website promosi Kota Semarang dengan fitur navigasi animasi GooeyNav bertema merah putih.

## 🚀 Fitur

- ✨ Animasi navbar interaktif dengan efek gooey
- 🎨 Tema merah putih (bendera Indonesia)
- 📱 Responsive design
- ⚡ Built with Vite + React

## 📋 Prerequisites

- Node.js (v16 atau lebih baru)
- npm atau yarn

## 🛠️ Instalasi

1. Clone repository ini:
```bash
git clone <repository-url>
cd Web_semarang
```

2. Install dependencies:
```bash
npm install
```

3. Jalankan development server:
```bash
npm run dev
```

4. Buka browser di `http://localhost:5173`

## 📦 Build untuk Production

```bash
npm run build
```

File hasil build akan ada di folder `dist/`

## 🎨 Struktur Project

```
Web_semarang/
├── assets/           # Gambar dan media
├── src/

│   ├── main.jsx     # Entry point

```
